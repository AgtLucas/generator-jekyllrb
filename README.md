# generator-jekyll

> A generator for Jekyll projects. yo, jekyll!

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator **locally**: `npm install generator-jekyll`
- Run: `yo jekyll`
