# generator-jekyllrb

> A generator for Jekyllrb projects. Yo, Jekyllrb!

## Getting started

- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator **locally**: `npm install generator-jekyllrb`
- Run: `yo jekyllrb`
